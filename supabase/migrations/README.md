# Padrão de Migrations - Fitcoach

Este diretório segue o padrão do Supabase CLI para versionamento de banco de dados.

## Como usar:

1. As migrations devem ser arquivos SQL numerados (ex: `202512240001_vulnerabilidade_fixed.sql`).
2. Utilize o Supabase CLI para gerenciar as mudanças no banco.
3. Evite rodar DDL direto no Console sem registrar aqui primeiro, para evitar "drift" de esquema.
